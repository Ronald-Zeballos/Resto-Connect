# Sprint 3 - Endurecimiento

## Objetivo

Preparar la aplicacion para crecer y operar de forma mas segura.

## Alcance propuesto

- Tests de integracion con `@SpringBootTest` y base efimera.
- Endurecimiento de seguridad JWT y manejo de errores.
- Auditoria de estados criticos de pedidos, pagos e inventario.
- Documentacion de despliegue y perfiles por ambiente.
- Ajustes de seeds, migraciones incrementales y compatibilidad futura con IA externa.

## Criterio de cierre

- Flujos criticos con cobertura integral.
- Base lista para CI/CD.
- Menor riesgo de regresiones al seguir creciendo.
